XMLDOC.Parser

<static> object XMLDOC.Parser
    XML Parser object. Returns an .Parser.node which is the root element of the parsed document.

    By default, this parser will only handle well formed XML. To allow the parser to handle HTML, set the XMLDOC.Parser.strictMode variable to false before calling XMLDOC.Parser.parse().

    Note: If you pass poorly formed XML, it will cause the parser to throw an exception.

XMLDOC.Parser.strictMode

<static> Boolean XMLDOC.Parser.strictMode
    Strict mode setting. Setting this to false allows HTML-style source to be parsed. Normally, well formed XML has defined end tags, or empty tags are properly formed. Default: true
