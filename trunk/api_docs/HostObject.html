Class HostObject
Extends HashContainer.

A host object is a container of components which provide some sort of action, be it rendering, collision detection, effects, or whatever. This way, an object can be anything, depending on it's components.

add

add(component)
    Add a component to the host object. The components will be sorted based on their type, and priority within that type. Components with a higher priority will be sorted before components with a lower priority. The sorting order for type is:
    • Input
    • Logic
    • Collision
    • Rendering
    Parameters:
      [BaseComponent] component - A component to add to the host
