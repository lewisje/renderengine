Class TouchInputComponent
Extends InputComponent.

A component which responds to touch events and notifies its HostObject by calling one of four methods. The HostObject should implement any of the following methods to receive the corresponding event:

  • onTouchStart() - A touch event started
  • onTouchEnd() - A touch event ended
  • onTouchMove() - A movement occurred after a touch event started
  • onTouchCancel() - A touch event was cancelled
Each function should take up to two arguments. The first argument is an array of Touch objects which represent each touch that occurred in the event. Some platforms support multi-touch, so each touch will be represented in the array. The second argument is the actual event object itself.
