Class CircleColliderComponent
Extends ColliderComponent.

An extension of the ColliderComponent which will check if the object's are colliding based on a bounding circle.

CircleColliderComponent

CircleColliderComponent(name, collisionModel, priority)
    For this component to function, a movement vector must be available on the host object by implementing the Object2D.getVelocity method which returns a Vector2D instance. Additionally the host object and collision object must implement the Object2D.getCircle method which returns a Circle2D instance.
    Parameters:
      [String] name - Name of the component
      [SpatialCollection] collisionModel - The collision model
      [Number] priority - Between 0.0 and 1.0, with 1.0 being highest
Method Detail

testCollision

Number testCollision(time, collisionObj)
    Call the host object's onCollide() method, passing the time of the collision and the potential collision object. The return value should either tell the collision tests to continue, or to stop.

    A circular bounding area collision must occur to trigger the onCollide() method.

    Parameters:
      [Number] time - The engine time (in milliseconds) when the potential collision occurred
      [HostObject] collisionObj - The host object with which the collision potentially occurs
    Returns:
      Number A status indicating whether to continue checking, or to stop
