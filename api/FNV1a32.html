Class FNV1a32

Implementation of FNV1a - a fast hash function. The FNV1a variant provides a slightly better dispersion for short (< 4 bytes) values than plain FNV1.

This implementation uses 32-bit operations, and the values returned from () are limited to the lower 32 bits.

Author: Andrzej Bialecki <ab@getopt.org>
