Static Class EngineSupport

A static class with support methods the engine or games can use. Many of the methods can be used to manipulate arrays. Additional methods are provided to access query parameters, and generate and/or read JSON.

filter

filter(array, fn, thisp)
    Calls a provided callback function once for each element in an array, and constructs a new array of all the values for which callback returns a true value. callback is invoked only for indexes of the array which have assigned values; it is not invoked for indexes which have been deleted or which have never been assigned values. Array elements which do not pass the callback test are simply skipped, and are not included in the new array.
    Parameters:
      [Array] array - The array to filter.
      [Function] fn - The callback to invoke. It will be passed three arguments: The element value, the index of the element, and the array being traversed.
      [Object] thisp - Used as this for each invocation of the callback.

sysInfo

 sysInfo()
    Gets an object that is a collation of a number of browser and client settings. You can use this information to tailor a game to the environment it is running within.
    • browser - A string indicating the browser type (safari, mozilla, opera, msie)
    • version - The browser version
    • agent - The user agent
    • platform - The platform the browser is running on
    • cpu - The CPU on the machine the browser is running on
    • language - The browser's language
    • online - If the browser is running in online mode
    • cookies - If the browser supports cookies
    • fullscreen - If the browser is running in fullscreen mode
    • width - The browser's viewable width
    • height - The browser's viewable height
    • viewWidth - The innerWidth of the viewport
    • viewHeight - The innerHeight of the viewport
    • support:
      • xhr - Browser supports XMLHttpRequest object
      • geo - navigator.geolocation is supported
      • threads - Browser supports Worker threads
      • sockets - Browser supports WebSocket object
      • storage:
        • local - localStorage object is supported
        • session - sessionStorage object is supported
        • global - gloablStorage object is supported
        • database - indexedDB storage is supported
      • canvas:
        • emulated - Canvas support emulated by FlashCanvas
        • defined - Canvas is either native or emulated
        • text - Supports text
        • textMetrics - Supports text measurement
        • contexts:
          • ctx2D - Supports 2D
          • ctxGL - Supports webGL
    Returns:
      An object with system information
