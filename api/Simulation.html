Class Simulation
Extends BaseObject.

A representation of a physical world. This object is used to introduce Box2dJS physics into your game by creating a world which supports the physical structures provided by Box2dJS. You will need to create a Simulation before you can utilize physics in a game.

See either "/demos/physics/" or "/demos/physics2" for examples of utilizing the Simulation object with rigid body components.
