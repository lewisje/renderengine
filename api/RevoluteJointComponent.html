Class RevoluteJointComponent
Extends BaseJointComponent.

A revolute joint which allows two bodies to revolve around a common anchor point in a Simulation.

setMotorTorque

setMotorTorque(torque)
    Set the torque which is applied via the motor, or to resist forces applied to it. You can use this value to simulate joint friction by setting the motor speed to zero and applying a small amount of torque. During simulation, the torque is applied directly to the joint.
    Parameters:
      [Number] torque - The amount of torque to apply
