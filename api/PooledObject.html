Class PooledObject

Pooled objects are created as needed, and reused from a static pool of all object types. When an object is created, if one is not available in the pool, a new object is created. When the object is destroyed, it is returned to the pool so it can be used again. This has the effect of minimizing the requirement of garbage collection, reducing cycles needed to clean up dead objects.
