Class FNV1Hash
Extends PooledObject.

A class for creating a single hash, or an evolving hash. Single hashes reset each time a new source string is provided, whereas the evolving hash will build upon previous hashes. The hash for a string will always be the same. Hashing a set of strings, in order, will always result in the same evolved hash. Uses the FNV1a32 hashing routine by default.
